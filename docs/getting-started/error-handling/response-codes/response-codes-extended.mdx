# Repsonse Codes (Extended)

## Additional Repsonse Codes

In addition to the Repsonse Codes page, DomPay's API generally returns a detailed error message and downstream errors.

| **HTTP Code** | **Description**                    |  **Message**                                                 |
| :------------ | :--------------------------------- | :----------------------------------------------------------- | 
| 400           | Internal Validation|<ul><li>validation_error</li><li>required_error</li><li>unfulfilled_condition</li></ul>|
| 400           | Gateway Validation|<ul><li>gateway_error</li><li>insufficient_funds</li><li>сredit_card_invalid_or_expired</li><li>transaction_declined</li><li>transaction_declined_hard</li><li>system_error</li><li>fraud_warning</li><li>invalid_transaction_details</li><li>unknown</li></ul>|
| 400           | Gateway Processing| <ul><li>gateway_error</li><li>insufficient_funds</li><li>сredit_card_invalid_or_expired</li><li>transaction_declined</li><li>transaction_declined_hard</li><li>system_error</li><li>fraud_warning</li><li>invalid_transaction_details</li><li>unknown</li></ul>|       
| 400 - 599     | Other             |<ul><li>unspecified_error</li></ul>|
