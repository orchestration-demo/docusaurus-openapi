## Additional Test Card Values

:::info
DomPay also allows you to set certain **expiry dates** when sending a transaction to the gateway, and expect a certain transaction response to be returned.

For example, a transaction with the expiry date of 04 / 27 returns the response EXPIRED_CARD.
:::

| **Expiry Date**        | **Expected Transaction Response**| 
|------------------------|----------------------------------|
| 01 / 39                | APPROVED                         |   
| 05 / 39                | DECLINED                         |
| 04 / 27                | EXPIRED_CARD                     |
| 08 / 28                | TIMED_OUT                        |
| 01 / 37                | ACQUIRER_SYSTEM_ERROR            |
| 02 / 37                | UNSPECIFIED_FAILURE              |   
| 05 / 37                | UNKNOWN                          |
