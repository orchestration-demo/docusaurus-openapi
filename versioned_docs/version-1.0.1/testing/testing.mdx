# Testing

## Overview

DomPay's Sandbox environment helps you confirm whether your integration works correctly and simulate real-time transactions.

## Test Scenarios

The Sandbox environment allows you to simulate several transaction scenarios using pre-assigned 'test cards'.

- Successful Transactions
- Declined Transactions
    - Do Not Honour
    - Expired Card
    - Time Out
    - System Error
    - Unspecified Failure
    - Unknown Failure ***(Downstream of DomPay)***
- Authentication Type Transactions
    - 3D-Secure 1.0
    - 3D-Secure 2.0
